# AI 批量去背 & 等比例縮放工具

這是一個基於 Streamlit 的網頁應用程式，可以批量上傳圖片，自動移除背景，並將圖片等比例縮放至指定大小。

## 功能特色

- **批量去背**：使用 `rembg` 強大的 AI 模型自動去除背景。
- **智能裁切**：自動偵測物件邊緣並裁切多餘空白。
- **等比例縮放**：根據設定的目標尺寸與佔比，自動縮放圖片 (置中)。
- **自訂背景**：可選擇透明背景或自訂顏色背景。
- **一鍵下載**：處理完成後可將所有圖片打包成 ZIP 下載。

## 安裝與執行

### 1. 安裝 Python

請確保您的電腦已安裝 Python 3.8 或更高版本。

### 2. 安裝相依套件

在終端機 (Terminal) 或命令提示字元 (CMD) 中執行以下指令：

```bash
pip install -r requirements.txt
```

### 3. 執行應用程式

執行以下指令啟動應用程式：

```bash
streamlit run app.py
```

啟動後，瀏覽器會自動開啟應用程式頁面 (通常是 `http://localhost:8501`)。

## 注意事項

- 首次執行時，程式會自動下載 AI 模型 (`u2net`)，約需 170MB，請耐心等候。
- 支援的圖片格式：PNG, JPG, JPEG, WEBP。
